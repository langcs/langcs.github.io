Sticky Footer布局

发表于 2017-08-18 | 分类于 CSS技巧

实现功能：

如果页面内容不够长，页脚区块固定在视窗底部；如果页面内容足够长，页脚区块根据内容的增多向下推送。

解决办法：

<html>
    <head>
        <title>Layout</title>
    </head>
    <body>
        <div class="content"></div>
        <div class="footer"></div>
    </body>
</html>

为内容区域添加最小的高度

content{
    min-height:calc(100% - footer的高度);
    box-sizing:border-box;
}

使用flex布局

body { 
    display: flex; 
    flex-flow: column; 
    min-height: 100%;
}
.content {
    flex: 1; 
}
.footer{
    flex: 0;      
}

在content的外面可以添加一个wrapper

html代码：

<html>
    <head>
        <title>Layout</title>
    </head>
    <body>
        <div class="wrapper">
            <div class="content"></div>
        </div>
        <div class="footer"></div>
    </body>
</html>

css代码：

html, body, .wrapper {
    height: 100%;
}
.wrapper {
    height: auto; 
    min-height: 100%;
}
.content {
    padding-bottom: footer的高度
}  
.footer {
    position: relative;
    margin-top: - footer的高度; 
    height: footer的高度;
    clear: both;
}

为了保证兼容性，在wrapper上添加新类clearfix

html代码：

<html>
    <head>
        <title>Layout</title>
    </head>
    <body>
        <div class="wrapper clearfix">
            <div class="content"></div>
        </div>
        <div class="footer"></div>
    </body>
</html>

css代码：

.clearfix{
    display: inline-block;
}
.clearfix:after {
    content: ".";
    display: block;
    height: 0;
    clear: both;
    visibility: hidden;
}
