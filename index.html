本地项目上传GitHub

发表于 2017-09-19 | 分类于 工具使用

1、先进入项目文件夹通过命令git init把这个目录变成git可以管理的仓库 

git init

2、把文件添加到版本库中，使用命令git add .添加到暂存区里面去，不要忘记后面的小数点“.”意思是添加文件夹下的所有文件

git add .

3、用命令git commit告诉Git，把文件提交到仓库

git commit -m '说明'

Sticky Footer布局

发表于 2017-08-18 | 分类于 CSS技巧

实现功能：

如果页面内容不够长，页脚区块固定在视窗底部；如果页面内容足够长，页脚区块根据内容的增多向下推送。

解决办法：

<html>
    <head>
        <title>Layout</title>
    </head>
    <body>
        <div class="content"></div>
        <div class="footer"></div>
    </body>
</html>
